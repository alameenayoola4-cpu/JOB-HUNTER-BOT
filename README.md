# Job Hunter Bot

A Telegram bot that automatically fetches developer job listings from 10+ platforms and sends them to you every morning.

---

## What It Does

This bot scans multiple job boards and sends you relevant developer job opportunities directly to your Telegram. No more manually checking dozens of websites - the jobs come to you!

---

## Job Roles

The bot finds jobs for:

- Frontend Developer
- Backend Developer
- Web Developer
- App Developer (Mobile)
- Full Stack Developer
- Software Engineer

---

## Job Sources

| Platform | Description |
|----------|-------------|
| RemoteOK | Remote tech jobs |
| Remotive | Remote developer jobs |
| Arbeitnow | European & Remote jobs |
| WeWorkRemotely | Remote work listings |
| TheMuse | Tech company jobs |
| Himalayas | Remote job aggregator |
| Reddit | r/forhire, r/remotejobs, r/hiring |
| HackerNews | "Who is Hiring" posts |
| WorkingNomads | Digital nomad jobs |
| JoBoard | Job aggregator |

---

## Schedule

Jobs are delivered daily at 8:00 AM UTC.
